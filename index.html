▸ Conversion tracking — install on any website
Works on any site — custom-built, web-designer built, Wix, WordPress, Shopify. Two ways to install: paste the snippet, or add the plugin file. Then mark what counts as a conversion.

        

          
          
        

        

          Marking the conversion (either option):

          · Add data-scanmap-convert to your lead form or buy button — fires automatically on submit/click

          · Or add data-scanmap-thankyou to the <body> tag of a thank-you page — fires on page load

          · Or call scanmapConvert() anywhere in your own code

          For the plugin file: upload scanmap-tracker.js to the site and add <script src="scanmap-tracker.js"></script> before </body>.
